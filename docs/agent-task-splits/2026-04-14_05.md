## Vong lap cap nhat tien do
- Khi Agent A hoan thanh, them dong log vao muc "Completion Log" theo format:
  - `A | <task> | DONE | HH:mm dd/MM/yyyy | <note>`
- Khi Agent B hoan thanh, them dong log vao muc "Completion Log" theo format:
  - `B | <task> | DONE | HH:mm dd/MM/yyyy | <note>`
- Khong xoa hoac sua dong log da co.
- Neu tiep tuc sang sub-task moi, them dong moi ngay ben duoi.

## Quy tac dong bo giua 2 agent
- A va B doc Completion Log truoc khi bat dau.
- Neu mot ben xong som, ghi ngay timestamp roi cap nhat note ngan.
- Ben con lai doc log de biet phan nao da xong va tranh dung chung file.
- Shared file chi cham khi bat buoc va co thong bao truoc.
